Dj MAD - AI Video Tool
Universal Windows installer

VERSION: {{VERSION}}

INSTALL (one file, any PC)
--------------------------
1. Download and run:  AI-Video-Tool-Setup-{{VERSION}}.exe

2. Choose install location (per-user or all users).

3. A setup console opens automatically and downloads:
   - AI engine (~1.6 GB)
   - WebView2 if needed
   - Best-fit AI model for your GPU and disk (or skip if hardware is limited)

4. The app launches when setup finishes.

REQUIREMENTS
------------
- Windows 10/11 64-bit
- Internet connection during first install
- NVIDIA GPU 8+ GB VRAM recommended (CPU mode available, slower)
- 15+ GB free disk minimum; 35+ GB for the default LTX model

RE-RUN SETUP
------------
Start Menu > AI Video Tool, or run setup.cmd from the install folder.
